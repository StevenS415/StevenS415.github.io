What is 5URV1V3?

5URV1V3 was the second game my group made in Game Design I. It is a 3D arena shooter game featuring a single melee enemy type that spawns in increasing numbers across waves. The goal was to create an engaging combat experience with progressively challenging enemy waves. It was also made in about 8 weeks.

What did I do?

Through the duration of this project, I was able to better understand Github and not rely on sending assets to my teammates, as well as familiarize myself with the Unity Asset Store. I found assets to use for enemies and was able to use Mixamo to animate them. I then set up the enemy's animator controller to properly transition between animation states.

Despite initially facing the same difficulty as Ransom Rabbit in regards to Github, I was able to learn how to properly use the version control software to be a more effective team member. This experience helped me to better understand how to navigate working with Unity.

What is Runaway Robots?

Runaway Robots was the game my group worked on in Senior Design I and II. We had 2 semester to make the game. It is a dimetric 3D grid-based strategy game where players navigate a maze, leading robots into hazards like electric fences and pits while avoiding dangers. Each cleared level advances to a more challenging, randomly generated level with faster robots.

What did I do?

This was the largest and longest project I had worked on. I was able to gain a greater understanding of Unity Editor features, as well as useful packages. I used the recorder package to get footage for the IOS App Store page. I also learned how to use GIMP, which I used to create the game icon and bomb sprite. I also created the music used in the game through the BeepBox web app.
